# Agentic-Design-Patterns 项目规则

## 环境配置

### Conda 环境
- **环境名称**: `Agentic-Design-Patterns`
- **重要**: 执行任何命令前，必须先激活环境：
  ```bash
  conda activate Agentic-Design-Patterns
  ```
- Python 路径: `D:\Anaconda\envs\Agentic-Design-Patterns\python.exe`



## 执行检查清单

在执行任何任务前，请确认：
